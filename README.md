## Project Overview
Campus Lost & Found is a simple queuing system that allows students and faculty to report lost items and help finding them.  
At the moment, most of the items lost are logged on WhatsApp or posted on notice boards, which is a mess and not very practical.  
We centralize lost & found posts, simplify searching and add a layer of security to claim requests.
